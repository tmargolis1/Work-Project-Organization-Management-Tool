This web chat program was created by Jacob Sacdalan in the Spring Semester of 2019.

The web chat utilizes the Pusher chatKit API for backend hosting. The chatKit
documentation for JavaScript can be found at: https://pusher.com/docs/chatkit/reference/javascript

The web chat application also utilized Node.js for a JavaScript run-time environment
and takes advantage of the React JavaScript library for building the UI.
The link for downloading Node.js is here: https://nodejs.org/en/download/

Some useful commands for setting up the modules needed for this app's environment follow:

Command Prompt/Powershell statements:


1. Install Dependencies: npm install
2. Install chatKit server SDK: npm install pusher-chatkit-server --save
3. Install chatKit client SDK: npm install --save @pusher/chatkit
4. Start application: npm start
